#### 安装
克隆下来后 `sudo make clean install` 安装过程可能你会缺少一些依赖,根据报错安装好依赖即可。<br/>
如果你使用了启动管理器,那么在登录时选择dwm即可。<br/>
如果你没有使用启动管理器,那么在 `.xinitrc` 写入一行 `exec dwm` 即可startx启动dwm<br/>
